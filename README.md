# Akks-e-Noor (عکسِ نور)

[![Website](https://img.shields.io/badge/Website-akks--e--noor.vercel.app-black?style=for-the-badge&logo=vercel)](https://akks-e-noor.vercel.app)

🔗 **Live Website:** [https://akks-e-noor.vercel.app](https://akks-e-noor.vercel.app)

---

**Akks-e-Noor** (Reflection of Light) is a formal community service digital platform dedicated to offering high-quality, beautifully designed Islamic posters, booklets, and educational media to the global community. 

This project is curated and presented as a free community service under the guidance and supervision of **Maulana Syed Imon Rizvi**. It is aimed at facilitating spiritual learning, promoting interfaith peace, and providing accessible Islamic resources to individuals, families, educational institutions, and community centers worldwide.

---

## 🌟 Vision & Mission

Our mission is to spread the beautiful, peaceful, and educational message of Islam through high-quality visual media and publications. By digitizing these materials, **Akks-e-Noor** ensures that anyone, anywhere, can download, print, and share these resources for religious events, educational study, or personal reflection—completely free of charge.

---

## 📋 Features

- **High-Resolution Islamic Media**: A rich collection of religious posters, calendar event graphics, and educational cards covering significant historical and seasonal dates (e.g., Muharram, Safar, Rabi' al-Awwal).
- **Free Publications & Booklets**: Digital access to printable booklets, such as the *Safar Booklet*, designed to provide insights into Islamic history, supplications, and guidelines.
- **Advanced Interactive Browsing**:
  - Filter content by category (e.g., Booklets, Seasonal Events, Historical Milestones).
  - Search functionality to find specific topics instantly.
- **Performance Optimized**:
  - **Progressive Image Loading**: Fast page rendering and smooth visual placeholder transitions for low-bandwidth users.
  - **Optimized Download Options**: Direct, high-quality downloads of individual posters and booklets.
- **Admin Dashboard**: A secure portal for administrators to upload new posters, manage content metadata, and update the repository dynamically.

---

## 🛠️ Technology Stack

The platform is built using modern web technologies to ensure speed, stability, and responsiveness:

- **Frontend Framework**: [TanStack Start](https://tanstack.com/start) with React 19 (Vite-powered SSR framework)
- **Styling**: Vanilla CSS with modern [Tailwind CSS v4](https://tailwindcss.com/) integration
- **Routing & State**: TanStack Router & TanStack Query for seamless page transitions and efficient data caching
- **Hosting & Serverless SSR**: Nitro server engine deployed on Vercel
- **Package Management**: Powered by Bun / npm

## 🚀 Getting Started

If you are a developer looking to run the project locally or contribute to the community service platform:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) (version 20+) or [Bun](https://bun.sh/) installed.

### Installation
Clone this repository and install the dependencies:
```bash
# Using bun (recommended)
bun install

# Or using npm
npm install
```

### Run Locally
Start the development server:
```bash
# Using bun
bun dev

# Or using npm
npm run dev
```
Open [http://localhost:8080](http://localhost:8080) in your browser to view the application.

### Build for Production
To create a production-ready build with Server-Side Rendering (SSR) optimized for deployment:
```bash
# Using bun
bun run build

# Or using npm
npm run build
```
